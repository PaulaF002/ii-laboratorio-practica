Preguntas
¿Qué es un framework?
Es un entorno de trabajo, o marco de trabajo es un trabajo conjunto estandarizado de conceptos, prácticas y criterios para enfocar un tipo de problemática particular que sirve como referencia.
¿Qué es bootstrap 5 y cuál es su origen?
Es un framework de código abierto disponible para HTML, CSS y JS desde finales del 2011 que se utiliza para diseñar páginas web adaptativos con un enfoque centrado en los dispositivos móviles de forma rápida y sencilla. Fue diseñador por Mark Otto y Jacob Thornton ambos en Twitter crearon este Framework para usarlo internamente en los proyectos de la compañía.
¿Qué es diseño responsive en páginas web?
Es una técnica de diseño web que busca la correcta visualización de una misma página en distintos dispositivos. Desde ordenadores de escritorio a tablets y móviles.
¿Qué son las Github pages?
Github es un sitio que permite subir repositorios de códigos para almacenarlo en el sistema de control de versiones. Se puede colaborar con proyectos de código, es sistema de código abierto lo que significa que cualquiera puede acceseder a tu código, usarlo, aprender y mejorarlo.
¿Cuál es la diferencia entre elemento y componente web?
Los elementos personalizados proporcionan una forma para que los autores creen sus propios elementos DOM con todas las funciones. Los componentes web son bloques de código que encapsulan la estructura interna de elementos HTML, incluyendo CSS y JavaScript, permitiendo así que el código se pueda volver a usar como se quiera en otras webs y aplicaciones.
¿Cuál es la importancia de documentar o comentar código?
La documentación se integra de todos los elementos que explican las características de un programa o sistema, y son necesarios para poder utilizarlo, operarlo o modificarlo. El código indica cómo funciona el programa, y la documentación indica porqué lo hace.
¿Por qué existe la sección de atribución en un sitio web?
asigna el valor de las conversiones teniendo en cuenta el modo en que los usuarios han interactuado con tus anuncios y han decidido convertirse en tus clientes. Este modelo de atribución usa los datos de tu cuenta para determinar qué palabras clave, anuncios y campañas contribuyen en mayor medida a que alcances tus objetivos de negocio.
